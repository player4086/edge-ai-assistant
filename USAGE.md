# AI 网页助手 — 使用指南

## 快捷键

| 快捷键 | 功能 | 说明 |
|--------|------|------|
| `Alt + A` | 打开助手 | 打开/切换到侧边栏 |
| `Alt + E` | 解释代码 | 选中网页上的代码后按此键，AI 自动解释 |
| `Alt + T` | 翻译文本 | 选中网页上的文本后按此键，AI 自动翻译 |
| `Alt + S` | 摘要页面 | 提取当前页面正文，AI 生成要点摘要 |

> **提示**：快捷键在 Edge 全局生效，无需先打开扩展。

## 五种操作方式

### 1. 浮动按钮（选中网页文本即出现）

在任意网页选中文本后，附近弹出 4 个按钮：

| 按钮 | 作用 |
|------|------|
| 解释 | 将选中代码发送给 AI 解释 |
| 翻译 | 将选中文本发送给 AI 翻译 |
| 摘要 | 提取当前页面正文发送给 AI 摘要 |
| 高亮 | 黄色荧光笔标记选中文本（刷新后保留） |

点击已高亮文本可取消高亮。

### 2. 右键菜单

| 菜单项 | 作用 |
|--------|------|
| AI 解释代码 | 解释选中的代码 |
| AI 翻译 | 翻译选中的文本 |
| AI 摘要页面 | 摘要当前页面内容 |

### 3. 侧边栏直接对话

点击扩展图标打开侧边栏 → 在输入框中输入问题 → 点击"发送"或按 `Ctrl + Enter`。

支持：粘贴代码、粘贴外文、粘贴网页内容，AI 自动识别并处理。

### 4. 快捷模板

点击输入框上方的模板按钮，Prompt 自动填入输入框：
- **代码审查** — 检查 Bug、安全问题、性能瓶颈
- **写注释** — 为代码添加详细注释
- **找Bug** — 找出错误并给出修复方案
- **优化** — 优化性能和可读性

可在"设置"中添加自定义模板。

### 5. 上传文件/图片

- **文件** — 上传代码文件（.js .py .c .cpp .java .rs .go .json 等 20+ 格式），自动以代码块发送
- **图片** — 上传图片（限 10MB），AI 可识别图片内容并描述

## 模式说明

| 模式 | 作用 | AI 的 System Prompt |
|------|------|---------------------|
| 自由问答 | 通用对话 | 自动判断：代码→解释，外文→翻译，网页→摘要 |
| 解释代码 | 专注代码分析 | 用中文解释功能、逻辑、输入输出、注意事项 |
| 翻译 | 多语言翻译 | 支持 8 种目标语言，保留技术术语准确性 |
| 摘要 | 内容精炼 | 3-5 个要点，突出关键信息 |

翻译模式支持 8 种目标语言：中文、English、日本語、한국어、Français、Deutsch、Español、Русский。

## 其他功能

### 主题切换
点击标题栏"主题"按钮在暗色和亮色之间切换。

### 导出对话
点击标题栏"导出"按钮，下载当前对话为 `.md` 文件。

### TTS 朗读
AI 回复右上角悬停出现 🔊 按钮，点击朗读回复内容（浏览器 Web Speech API，无需联网）。

### API 设置
点击标题栏"设置"按钮，配置：
- **API Key** — 你的 DeepSeek API Key
- **API Base URL** — 默认 `https://api.deepseek.com/anthropic`
- **Model** — 默认 `deepseek-v4-pro[1m]`

## 安装与更新

### 首次安装
1. Edge 打开 `edge://extensions`
2. 开启"开发人员模式"
3. "加载解压缩的扩展" → 选择项目文件夹

### 更新
1. 拉取最新代码
2. `edge://extensions` → 找到本扩展 → 点击刷新按钮
